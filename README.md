# your-inbox
Will be made soon! Using: Visual Studio Code, HTML, CSS and JavaScript.

View storyboard [here](https://docs.google.com/presentation/d/15ksF_2PTSywoEOhdTFEGVXB-Q3EdV7P9_DGxWM1xduc/edit?usp=sharing), thanks. 

**The problem**:
About 80% of people state that work related emails makes them anxious, over half regularly feel stressed about it (Team, 2025). This can also cause an unhealthy amount of procrastination and avoidance. 

**The solution**:
Gamified version of inbox clearing to be made. This allows users to practice checking, refresing and clearing their randomised emails in a low-stakes, satisfying environment. Users can create an account, and login regularly to practice clearing their inbox, allowing them to slowly face and overcome their fears.
